Code Tutor and Keyer

This is a combinatin of cw totor and a keyer. I hope it helps you with CW and make code enjoyable. The keyer is for rigs that don't have a keyer built in but do generate side tone

Dit, dah, and spacing times are calculated based on the data entered in the speed and Farnsworth speed fields according to this paper: http://www.arrl.org/files/file/Technology/x9004008.pdf. Letter (3 dit lengths) and word (7 dit lengths) spacings as defined by the Farnsworth algorithm and are stretched by the factors you specify. Experiment to see what is comfortable for you.
